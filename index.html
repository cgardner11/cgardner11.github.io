Resources

  • PEXELS ~ Pexels is a nifty source for some of the best free stock photos around. Come on, a website is more dynamic and interesting when images are used purposely, because we are after all...very visually stimulated. So finding free valuable resources for high-quality, stock images comes in quite handy for a web developer, and it helps keep the costs of creating the site down, which will make everybody a happy camper.
  • W3SCHOOLS ~ Sometimes we just get stuck on how to accomplish a certain action when writing code. I found w3Schools an extremely helpful site that gives you definitions for HTML and CSS, it allows you a real-time, online, on-the-spot practice view using given code examples that will just make your life easier.
  • STACK OVERFLOW ~ Stack Overflow is an online community type blog where millions of programmers congregate to help each other with questions and answers. This site is chalked full of free tips shared by other developers, with fun tricks, and advice on sticky issues when writing code.
  • CoDrops ~ Codrops is a design and development blog with articles and tutorials about the latest web tips and trends. There's a valuable "Tutorial" section for how-to's on web development techniques, a fun "Playground" for plug-ins and helpful articles, a "Blueprints" section that has sample website concepts and layouts, and an extensive CSS reference section.
